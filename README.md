## 개요
- 판타지 스포츠와 베팅 시스템을 결합한 웹 애플리케이션입니다.
- 유저가 원하는 11인의 로스터로 베팅을 하여 당일 선수들의 경기 성적에 따라 배당금을 획득할 수 있는 시스템입니다.
- 개발 기간 : 2021년 4월 12일 (월) ~ 2021년 4월 30일 (금) 약 3주
